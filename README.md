# AppointmentWebApp

## Description  
AppointmentWebApp is a full-stack web application designed for scheduling, managing and tracking appointments easily and efficiently. The system enables users to view available slots, book new appointments, edit or cancel existing ones, and get real-time status updates on their bookings.

## Features  
- User registration & authentication  
- Browse available appointment slots  
- Book, edit or cancel appointments  
- View upcoming and past appointments in dashboard  
- Real-time updates for appointment status  
- Clean modular architecture with frontend + backend separation  

## Tech Stack  
- **Frontend**: React.js (create-react-app / Vite)  
- **Backend**: Node.js, Express.js  
- **Database**: MongoDB  
- **APIs**: RESTful endpoints for appointments & user management
